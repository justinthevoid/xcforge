## Bootstrap

```bash
cd Web
bun install
```

## Validation Commands

```bash
cd Web
bun run typecheck
bun run lint
bun run validate
```

## Local Route Checks

Run the dev server:

```bash
cd Web
bun run dev
```

Then verify these routes:

- Marketing route: `/`
- Docs route: `/docs`
- Docs depth route: `/docs/getting-started`

## If Setup Fails

Run the environment checker:

```bash
cd Web
bun run doctor
```

For full remediation steps, see the Troubleshooting section in `README.md`.
